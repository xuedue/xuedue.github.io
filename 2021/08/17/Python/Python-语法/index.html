sort & sorted

  • sort
    list内置的sort()函数(只适用于list)，本身不会返回新的list，会改变原先的list的顺序(节省空间，提高效率)
  • sorted
    sorted()函数适用于任意可以迭代的对象排序，不会改变原对象，返回一个新的可迭代对象
